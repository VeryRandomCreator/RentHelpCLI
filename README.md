# Rent Help CLI Verification Tool
This is the CLI tool verifying pdf-certificate-signature bundles generated by [RentHelp](https://github.com/VeryRandomCreator/RentHelp/).

## How it works: Secure PDF Generation Process
1. Loads files
2. Verifies that the signature is associated with the certificate chain
3. Verifies that the signature provided is associated with the pdf.
    - Ensures the pdf is unmodified
4. Verifies certificate chain its generated through Android's TEE
5. Verifies the TEE generating the certificate chain is associated with the official apk of [RentHelp](https://github.com/VeryRandomCreator/RentHelp/).
    - Compares the value found in the certificate chain to the public value displayed in the build action on [RentHelp](https://github.com/VeryRandomCreator/RentHelp/)
6. Prints the block hash and timestamp associated with it. 

## How to Use
Download and compile CLI
